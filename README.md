# Messi X Cristiano

A single-page tribute site exploring the Messi vs Ronaldo football rivalry with interactive carousels, biographies, and a comparative timeline.

## Features

- **Hero carousel** — Swipeable flip cards with circular/infinite scrolling
- **Duo comparison** — Side-by-side Messi & Cristiano with interactive divider
- **Biographies** — Messi and Ronaldo multi-page bios (Early Life, Barcelona Era, Legacy)
- **Timeline carousels** — Year-based image carousels for both players

## How to run

1. Clone the repo
2. Ensure the `assets/` folder contains the required images (hero, duo, bio, timeline)
3. Open `index.html` in a browser, or serve the folder locally:
   ```bash
   npx serve .
   # or: python -m http.server 8000
   ```

## Tech

- HTML, CSS, JavaScript (single file, no build step)
- Google Fonts (Playfair Display)
